# Agentic Engineering - Interactive Deck

Interactive presentation deck explaining the Agentic Engineering approach with 12 specialized AI agents.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Features

- 13 interactive slides
- Keyboard navigation (arrow keys, spacebar)
- Click navigation dots to jump to any slide
- Explains the Worker-Governance Pattern
- Shows all 12 agents and their roles
- Live code examples

## Slides Overview

1. **Title** - Introduction
2. **Problem** - Traditional vs Agentic development
3. **Pattern** - Worker-Governance explained
4. **Worker Agents** - 6 execution agents
5. **Governance Agents** - 6 quality/direction agents
6. **Install** - Quick start commands
7. **Workflow** - 4-phase agentic workflow
8. **Example 1** - Defining requirements
9. **Example 2** - Architecture and build
10. **Example 3** - Quality gates
11. **Balance** - Why the pattern works
12. **Resources** - Links and docs
13. **Thanks** - Closing

## Navigation

- **Arrow keys**: Left/Right to navigate
- **Spacebar**: Next slide
- **Click dots**: Jump to specific slide
- **Buttons**: Prev/Next

## Tech Stack

- React 19
- Vite 7
- Tailwind CSS 4
- Lucide React (icons)
